# 📖 使用方法

#### 📝 步骤

#### 1. 克隆本仓库

```
git clone https://github.com/dddinmx/hxs-downloader.git
```

#### 2.安装依赖

```
pip3 install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

#### 3.运行
```
python3 hxs-downloader.py
```
